# Attend

Веб-приложение для ведения журнала посещаемости групп по плаванию.

## Возможности

- создание групп с расписанием по любым дням недели;
- создание бассейнов или мест работы и групп внутри них;
- указание времени тренировки;
- ручная сортировка групп перетаскиванием;
- добавление учеников в группу с годом рождения с выбранного месяца;
- редактирование имени и года рождения ученика;
- удаление ученика с выбранного месяца без потери истории в предыдущих месяцах;
- полное удаление ученика из всех месяцев вместе с отметками;
- журнал посещаемости в формате школьной таблицы;
- отметки `+`, `-` или пустая ячейка;
- сохранение данных в браузере через `localStorage`;
- экспорт текущего месяца в CSV.

## Запуск

Откройте `index.html` в браузере.
