# Attend

Веб-приложение для тренера по плаванию: бассейны, группы, ученики и журнал посещаемости.

## Что уже есть

- авторизация через Supabase (email/password);
- бассейны и группы с расписанием по дням недели;
- ученики с годом рождения;
- журнал с отметками `+`, `-`, пусто;
- быстрые отметки на главной;
- перенос ученика между группами с историей;
- резервная копия / восстановление (JSON);
- экспорт текущего месяца в CSV;
- синхронизация данных с Supabase.

## Локальный запуск

```bash
npm install
npm run dev
```

Приложение откроется на `http://127.0.0.1:5173`.

## Деплой на Vercel

1. Импортируйте репозиторий в Vercel.
2. Framework preset: `Vite`.
3. Build Command: `vite build`.
4. Output Directory: `dist`.
5. Добавьте переменные окружения проекта:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`

Важно: в текущей версии ключ и URL уже заданы в `index.html`. Для production лучше вынести их в env и читать через `import.meta.env`.

## Supabase

Для корректной работы должны быть таблицы:

- `profiles`
- `pools`
- `groups`
- `students`
- `attendance`

И включены RLS-политики для доступа только к данным текущего пользователя.
